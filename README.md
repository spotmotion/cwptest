# Cavalry Web Player

## Quick Start

### macOS
1. Open Terminal
2. `cd /path/to/cavalry-web-player`
3. `python3 serve.py`
4. Open http://localhost:8000 in your browser

### Windows
1. Open Command Prompt or PowerShell
2. `cd \path\to\cavalry-web-player`
3. `python serve.py` (or `python3 serve.py`)
4. Open http://localhost:8000 in your browser

## Documentation

For full documentation, see https://docs.cavalry.scenegroup.co/web-player/.
