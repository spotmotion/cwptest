About this Demo

This is the minimal example of the Cavalry Web Player. It demonstrates the basic setup required to load and play a Cavalry scene in the browser.

This example includes only the essential functionality: scene loading, playback controls, and frame scrubbing. It serves as a starting point for building more complex experiences.

Please note, assets will not work in this simple player.
